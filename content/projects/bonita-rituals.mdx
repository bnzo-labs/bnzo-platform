Bonita Rituals is a wellness brand with strong editorial DNA. The previous site was Shopify out of the box — functional, but it looked like every other Shopify store. The brand needed something that felt intentional.

The split architecture made sense immediately. Shopify owns commerce: cart, checkout, inventory, payments. Sanity owns editorial: ritual guides, brand story, ingredient sourcing. Next.js renders both in a single coherent experience.

The agent piece was bundle copy. Given a list of SKUs with metadata — ingredients, benefits, scent profiles — the agent generates bundle descriptions in the brand's voice. We reviewed the first batch manually. It passed. We've stopped reviewing.

LCP went from 3.2s to 0.9s. That came from image optimization and ISR — not from any clever trick. The conversion lift was real but attribution is messy; the redesign and the copy improvements happened simultaneously.

Sanity preview was the time sink. The preview URL scheme required undocumented configuration that took half a day to find. That's in our notes now.
